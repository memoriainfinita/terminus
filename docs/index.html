Referencia API data-* attributes

Atributos HTML

AtributoTipoDefaultDescripción
data-theme String "dark" dark · matrix · ocean · amber
data-prompt String "user@host:~$" Texto del prompt
data-welcome String null Mensaje inicial al cargar
data-commands JSON {} Comandos como strings. Para lógica usa la API JS.

API JavaScript

MétodoDescripción
t.setPrompt(str)Cambia el prompt en tiempo real
t.setCommands({cmd: fn|str})Añade o sobreescribe comandos
t.addOutputHTML(html)Imprime HTML en la terminal
t.clear()Limpia el output
t.play(steps)Reproduce una secuencia animada
t.readline(prompt)Espera input del usuario (Promise)
t.lock() / t.unlock()Bloquea/desbloquea el input

Acceso a la instancia

// Vía element.terminalComponent
const t = document.getElementById('mi-terminal').terminalComponent;

// Vía array global
const t = window.terminalInstances[0];

// Comandos como funciones (lógica avanzada)
t.setCommands({
  'ping': (args, term) => {
    const host = args[0] || 'localhost';
    term.addOutputHTML(`<span style="color:#00ff88">Reply from ${host}: time=1ms</span>`);
  }
});

Comandos JSON: valores string


{
  "help":    "Lista todos los comandos disponibles",
  "about":   "Información sobre este terminal",
  "status":  "Sistema: Online\nCPU: 45%\nRAM: 2.1GB/8GB",

Consideraciones de Seguridad

              
No incluyas información sensible en data-commands. Si conectas comandos a una API, valida los inputs del lado servidor.

Preguntas FrecuentesFAQ

          

        
        

          

            

              
¿Qué tan fácil es implementar TERMINUS?

              +
            

            

              
Solo 2 líneas de código: CSS y JS. Auto-inicialización automática, sin configuración adicional. Literalmente copias dos archivos y funciona inmediatamente.

            

          


          

            

              
¿Puedo personalizar temas y colores?

              +
            

            

              
4 temas predefinidos (matrix, ocean, amber, dark) + temas personalizados via CSS variables. Control total sobre colores, fuentes y animaciones.

            

          


          

            

              
¿Cómo agregar comandos personalizados?

              +
            

            

              
Atributo data-commands con JSON o configurador interactivo incluido. Soporte para comandos simples, objetos complejos y respuestas dinámicas.

            

          


          

            

              
¿Es compatible con móviles?

              +
            

            

              
Completamente responsive. Funciona en desktop, tablet y móvil con teclado virtual optimizado y gestos táctiles incluidos.

            

          


          

            

              
¿Necesita dependencias externas?

              +
            

            

              
Cero dependencias. Vanilla HTML/CSS/JS puros. Bundle: 12.4 KB minificado. No requiere jQuery, React ni ninguna librería externa.

            

          


          

            

              
¿Cómo distribuir en producción?

              +
            

            

              
CDN jsDelivr, GitHub Pages o descarga directa. Archivos minificados incluidos en el repositorio.
